# Online Coffee Store
An online store for a coffee shop.

## Features
This store has the following features;
### General Features
- Menu listings
- Login/Signup features
- View the store details (About and Contact)
### Admin Features
- Manage Users
- Manage Menu Listings
- Keep track of orders

## Technologies
- Django
- Sqlite3
